# About

A battlebot made from the SSP kit provided by Just Cuz Robotics, utilizes the vertical spinner modification.

Kit: https://justcuzrobotics.com/products/ssp-robot-kit 
Additional Modification: https://justcuzrobotics.com/products/vert-module-ssp-2


It contains the receiver FrSky X8R, which is connected to the FrSky Taranis X9D+ Transmitter.
The code contained in this repository is tested on OpenTX on the transmitter.
